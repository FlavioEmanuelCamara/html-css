Imagens dinâmicas

Perceba que essas imagens vão mudar conforme mudamos o tamanho do navegador.

  • Um "P" amarelo = imagem pequena
  • Um "M" rosa = imagem media
  • Um "G" Verde = imagem grande

Essas tags <picture> e <source> servem justamente pra enquadrar as imagens ao tamanho do navegador do usuário que irá acessar o site, seja acessando por computadores, tablets ou celulares. Não utilizar esses recursos significa que um usuário, por exemplo, que usa um celular, sofrerá com rolagem horizontal no navegador, pois estará acessando uma imagem para uma tela bem maior, além de sofrer com lentidão, pois a imagem é desproporcionalmente pesada para o seu dispositivo.
